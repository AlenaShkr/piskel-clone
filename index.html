-clone

Piskel-clone is the project of free online editor for animated sprites & pixel art

Author strive create application similar Piskel:

  • Canvas sizes

    you can choose size canvas : 32x32 units / 64x64 units / 128x128 units

  • Tools: pen, all paint bucket, eraser, color picker

    you can drawing by pen, paint all pixels of the same color, clean picture, define element colour and set it

  • Color select

    you can choose painting color (primary and secondary)

  • Preview

    In small animation preview window in the top right corner can look at animation and in full screen mode

  • Frames

    Ability to add/delete a frame,ability duplicate a frame
